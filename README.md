## 🇬🇧 ENGLISH

> Feature-rich clans system for **NG-PocketMine-MP** servers

[![API](https://img.shields.io/badge/API-5.39.0%2B-blue?style=flat-square)](https://github.com/NetherGames/PocketMine-MP)
[![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)](LICENSE)


---
### ✨ Features
| | |
|---|---|
| 💰 **Banking System** | Deposit & withdraw funds, only leader can withdraw |
| 👥 **Invitation System** | Invite players, accept/decline invitations |
| 🏆 **Ratings** | Top clans by power & balance |
| ⚔️ **Member Management** | Kick members, leader permissions |
| 🎨 **Beautiful UI** | SimpleForm, ModalForm, CustomForm — no external deps |
| 💵 **Paid Creation** | Configurable price via EconomyAPI |

---

### 📋 Commands

| Command | Description |
|---------|-------------|
| `/c create <name>` | Create a clan |
| `/c leave` | Leave the clan |
| `/c invite <nick>` | Invite a player |
| `/c accept` | Accept invitation |
| `/c members` | Member list |
| `/c info [name]` | Clan info |
| `/c bank` | Bank management |
| `/c deposit <amount>` | Deposit to bank |
| `/c withdraw <amount>` | Withdraw from bank |
| `/c powertop` | Power top |
| `/c balancetop` | Balance top |

---
### 🔧 Requirements
- NG-PocketMine-MP API **5.39.0+**
- [EconomyAPI](https://github.com/poggit/EconomyAPI)
